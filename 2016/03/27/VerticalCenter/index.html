日志CSS居中

这是新的开始，我用hexo创建了第一篇关于网页元素CSS居中【水平居中、垂直居中】。

>lz常常强调有图才有真相，如果没有图，最少也要有码吧，这个有点为难了，hexo的东西
还没有吃透，图片怎么引用路径，怎么显示还不会。。。有没有哪位高人指点一下！

>昔年有狂客 号尔谪仙人 笔落惊风雨 诗成泣鬼神 ——》我最喜欢的两位诗人
    写了半天的博客 突然发现有些东西都还没有弄清楚，感觉迷糊了
    水平居中：相对于容器来说 元素排版左右两边的距离相等
    垂直居中：网上没有查到对应的说法，我也是醉了；我们反过来想一想：
    把水平居中的那个概念拿过来用一下，水平修改为竖直方向 ，相对于容器上下距离一样，
    这里强调了一个容器的概念：可以理解为父级元素，就近原则啥的都用上吧，还待考证
    水平垂直居中：这个移动端比较常用，具体可以在网上看看

水平居

.compact
//水平居中：行内元素解决方案
//只需要把行内元素包裹在一个属性display为block的父层元素中，并且把父层元素添加如下属性即可
>.parent {
    text-align:center;
}
//水平居中：块状元素解决方案
>.item {
    /* 这里可以设置顶端外边距 */
    margin: 10px auto;
}

垂直居中

.compact
//水平垂直居中：未知高度和宽度元素解决方案
>.item{
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);  /* 使用css3的transform来实现 */
}
